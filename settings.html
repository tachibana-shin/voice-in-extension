Voice Commands

A voice command maps a phrase to a predefined block of text or an action. Think of voice commands as shortcuts to help you do your everyday tasks more efficiently. For example, the voice command 'insert name' ‚Üí 'John Snow' means that when you dictate insert name, John Snow will be inserted.

Voice In comes with a built-in list of voice commands for punctuation marks, line breaks, common actions, etc. Check-out the full list here.

To edit an existing voice command, simply delete it by clicking on icon for that voice command. Then add it again.
